# VS Code 小技巧

## 快速定位

- `Ctrl+P`：按文件名快速打开文件。
- `Ctrl+Shift+P`：打开命令面板。
- `Ctrl+Shift+F`：在整个工作区搜索文本。
- 在 `Ctrl+P` 中输入 `@`：跳转到当前文件中的符号。
- 在 `Ctrl+P` 中输入 `:` 加行号：跳转到指定行。

## Markdown

- `Ctrl+Shift+V`：打开 Markdown 预览。
- `Ctrl+K V`：在编辑器侧边打开 Markdown 预览。
- 使用相对路径插入本地图片，移动仓库后链接仍然有效。

## GitHub Wiki

- Wiki 页面就是 Markdown 文件，可以直接在 VS Code 中编辑。
- `Home.md` 是默认首页。
- `_Sidebar.md` 和 `_Footer.md` 分别控制侧栏与页脚。
- Wiki 使用独立的 `.wiki.git` 仓库，提交不会出现在主仓库历史中。
